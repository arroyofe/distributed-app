Presentación sucinta del proyecto.

Este proyecto contiene un frontend, un backend, ambas gestionadas con java una base de datos Mysql con datos de usuarios,items y pokemones, y dos aplicaciones python: una de gestión de pokemones y la otra hace de puente con frontend para la gestion de items. Se ha utilizado springboot, thymleaf, nginx, bootstrap para la realización del proyecto.
